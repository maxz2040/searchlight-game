# Welcome to Wheres-Waldo 👋

[Permalink: Welcome to Wheres-Waldo 👋](https://github.com/jwaddell10/Wheres-Waldo#welcome-to-wheres-waldo-)

Fun game based on classic Where's Waldo game. User has 3 map choices (easy, medium, hard). See if you can get the new high score on the leaderboard!

## Live Site:

[Permalink: Live Site:](https://github.com/jwaddell10/Wheres-Waldo#live-site)

[https://wheres-waldo-rho.vercel.app/](https://wheres-waldo-rho.vercel.app/)

## Features:

[Permalink: Features:](https://github.com/jwaddell10/Wheres-Waldo#features)

1\. 3 maps--easy, medium, and hard--to allow for variety in gameplay

2\. Visually appealing design and UI

3\. Leaderboard enables users to save their score

## Tech Stack:

[Permalink: Tech Stack:](https://github.com/jwaddell10/Wheres-Waldo#tech-stack)

MongoDB, Express, React, Node (MERN Stack Application)
